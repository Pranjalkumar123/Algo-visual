# AlgoVisual

A C++ console-based visualization tool for basic algorithms like sorting and searching.

## 🚀 Features
- Bubble Sort Visualization
- Quick Sort Step-by-step
- Linear & Binary Search Flow
- Menu-driven terminal interface
